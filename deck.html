üìò How to Use Deck Review Archive

This page explains how to use the Deck Review Archive system ‚Äî how to build decks, save them, copy/paste them, and request reviews.

1. Login

Enter your name in the Your Name field and press Enter. Your personal deck storage will be created automatically.

2. Build Your Deck

There are 8 slots in the deck builder:

  • Top-left 2 slots: Evo cards only
  • Top-right 2 slots: Hero cards only
  • Bottom 4 slots: Regular cards

Click a card in the card list to place it into the selected slot.

3. Card List Tools

  • Sort: Sort cards by Name / Elixir / Rarity
  • Card Size: Adjust the display size of the card icons
  • Average Elixir: Automatically shows the deck's average elixir cost

4. Submit Your Deck

Press the pink Submit button to save your deck into My Decks.

5. Copy Deck

The Copy Deck button copies your current 8-card deck into the clipboard as a JSON array. Example: 

["Hog","Knight","IceSpirit","Log","Fireball","Tesla","Musk","Cannon"]

6. Paste Deck (Auto Load)

If your clipboard contains a JSON deck array, press Paste Deck to load it instantly into the builder. This allows fast deck sharing and rebuilding.

7. My Decks

Press My Decks at the top to open your stored deck list. There you can:

  • View saved decks
  • Copy them
  • Delete them

8. Request Review

Switch the mode to Request Review and press Submit to send a deck review request. These requests appear in the admin panel.
